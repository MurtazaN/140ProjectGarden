Project 1 - Designing a Flower Bed

Project Overview

This project will give you experience on the use of:

  1. Java primitives int and float
  2. Arithmetic operations
  3. Using Scanner to get user input
  4. Using println statement
The goal of this project is to gain experience with mathematical manipulation of numeric values. You will calculate areas and volumes to determine materials required for a gardening project. The basic design consists of prompting the user for information, receiving information, processing that information, and then displaying the result. This project will provide you with the opportunity to put together nearly all of the material you have learned so far to create a single program. Most of what you are asked to do is very similar to what you did in the lab1 and lab2 and in-class work, so look back at those labs and class examples for ideas. You will be writing code for one method - main. Refer to Lab1 for a sample program with only the main method.

Background

Programs are good at performing routine mathematical calculations. By way of illustration, you will write a program to calculate the materials needed for an ornamental garden according to the design given here . In this design, the blue areas represent flowerbeds and the yellow areas are filled with stone, mulch, or other fill material. The garden is a perfect square. The four outer flowerbeds are congruent semicircles and the central flowerbed is a perfect circle.

Project Description and Specification

Your program should prompt the user for the following information:
  1. The side length (in feet) of the finished garden.
  2. The recommended spacing (in feet) between plants.
  3. The depth (in feet) of the flowerbeds.
  4. The depth (in feet) of the filled areas
Next estimate the number of plants and the amount of fill and flowerbed soil needed.

Finally, it should report the following quantities needed for the garden:

  1. Number of plants for each type of flowerbed (semicircle and circle) and total number of plants for the garden.
  2. Cubic yards of soil for each type of flowerbed (semicircle and circle) and total cubic yards of soil for the garden.
  3. Total cubic yards of fill material for the garden.

Project Notes

To clarify the problem specifications, I provided at the end of this document a snapshot of interaction with the already written program. The formula for the area A of a circle is A = π r2, where r denotes the radius. For this calculation, you should use the value of π provided in the Java's Math package. Simply use Math.PI. Math.PI gives you (a close approximation to) the value of π. To estimate the number of plants for a flowerbed, divide the area of the flowerbed by the area needed per plant (the square of the recommended distance between plants) and then truncate this result. To truncate a float, convert it to an int using casting which implicitly truncates the float. Be careful, the number of plants in a semicircle may not be exactly half of the plants in a full circle. Assume that the user only inputs numbers, that is, you don’t need to do error checking for bad input. Remember to convert fill to cubic yards. Here is an outline of your program:
  1. Prompt for input (use Scanner class's nextFloat method)
  2. Do some mathematical calculations using the input numbers to get results.
  3. Print the results (use System.out.println)
